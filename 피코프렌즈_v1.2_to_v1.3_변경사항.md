# 피코프렌즈 화면설계서 v1.2 → v1.3 변경사항

**문서 버전**: v1.2 → v1.3
**변경일**: 2025-10-21
**분석일**: 2025-10-27

---

## 변경사항 요약

v1.3은 Confluence 문서 내용을 반영하여 v1.2의 일부 기능을 개선한 버전입니다.
주요 변경사항은 **프론트엔드의 설문조사 항목 개선**과 **백오피스의 약국 방문 히스토리 관리 기능 추가**입니다.

---

## 1. 프론트엔드 변경사항

### PF-020: 메모(특이사항) - 약국 방문 설문 개선

#### 변경 전 (v1.2)
설문 항목이 일반적으로 정의되어 있었음

#### 변경 후 (v1.3)

**리포트 기능:**
- 구글 설문 데이터 불러오기 기능 유지
- 상세 설문 항목 구체화

**1. 약국 기본 정보**
- **약사님 성함** (필수) - OOO (텍스트 직접 입력)
- **약사님 연락처** (선택) - 010-1234-1234 (번호 형식)

**2. 약국 운영 및 애로 사항**

**온라인을 이용 현황** (필수)
- 체크박스 다중 선택:
  - ✓ HMP홈(온라인판)
  - 새로맘
  - 닥삼
  - 바로팜
  - 팜스트리트
  - 광중삼
  - JW삼
  - 유한몰
  - 기타 (텍스트 직접 입력)

**주력 판매 상품군** (필수)
- 체크박스 다중 선택:
  - 전문의약품
  - 일반의약품
  - 의약외품
  - 건강 기능 식품
  - 의료 기기 및 기타

**주력 판매 제품**
- OOO (텍스트 직접 입력)

**가장 시급하게 필요한 제품 또는 서비스** (필수)
- OOO (텍스트 직접 입력)

**3. 피코프렌즈 반응**

**피코픽 혜택 안내 후 반응** (필수)
- 라디오 버튼 단일 선택:
  - ○ 매우 좋음
  - ○ 좋음
  - ○ 보통
  - ○ 나쁨
  - ○ 기타 (텍스트 직접 입력)

**현장활동 요약 및 다음 단계 계획** (필수)
- OOO (텍스트 직접 입력)

#### 개선 내용 분석

**각 항목별 피코프렌즈가 직접 작성**
1. **약사님 약사 이름, 약사 연락처를** 약국 연락처는 기재된 것이고, 약사는 개인 정보로 약국 대상으로 한 설문
2. **피코프렌즈가 방문한 약사가 나타나고, 회원가입 시 관련 기록 확인가 나타나며, 회원가입한 경우는 가입완사 없는을 표기**
3. **피코프렌즈가 작성한 설문조사는 클릭 시, 설문 조사 작성 내용을 확인 할 수 있어야 함. (1-1)**
4. **피코의 경우 관리자가 작업 텍스트를 입력 및 수정**

**다음페이지 계속**

#### Description 설명 항목

1. **리포트 기능**
   - 구글 설문에 등록된 엑셀 파일의 약국 정보들을 불러옴
   - 리포트는 다음번 업무 배정 시에도 누적

2. **각 항목별 피코프렌즈가 직접 작성**
   - 약사님 이름 (단답형), 약사님 연락처 (단답형, 선택형)
   - 설문 조사 데이터는 관리자에서 출제 가능

3. **약사님 설문 (텍스트 입력)**
   - 1. 약사님 설문 (단답, 텍스트 입력)
   - 2. 약사님 연락처 (번호 형식)
   - 온라인을 이용 현황(체크박스, 중복 가능, 기타(텍스트입력))
   - 3. 주력 판매 상품군(체크박스, 중복 가능, 기타)
   - 4. 주력 판매 상품군(텍스트 직접 입력)
   - 5. 가장 시급하게 필요한 제품(텍스트 입력)

4. **마케팅/홍보 관련 고민(대리박스, 기타 텍스트 입력)**
   - 6. 마케팅/홍보 관련 고민(대리박스, 중복 가능, 기타(텍스트 입력))
   - 7. 피코프렌즈 반응(라디오박스, 기타 텍스트 입력)
   - 8. 현장활동 요약 및 다음 단계 계획

## 4. 주요 개선 포인트

### 4.1 설문조사 상세화
- 온라인 이용 현황을 구체적인 플랫폼 목록으로 제시
- 피코프렌즈 반응을 2단계(긍정/부정)에서 5단계로 세분화
- 현장활동 요약 및 다음 단계 계획 필드 추가로 방문 기록 강화

### 4.2 약국 방문 히스토리 관리 강화
- 약국별 모든 방문 이력을 히스토리로 관리
- 피코프렌즈별 방문 기록 추적 가능
- 설문조사 내용을 언제든 재확인 가능
- 관리자가 비고에 메모 추가 가능

### 4.3 데이터 관리 편의성 향상
- 약국 정보와 방문 기록을 통합 관리
- 회원가입 전환 여부를 히스토리에서 확인 가능
- 피코프렌즈 활동 성과를 약국 단위로 추적 가능

## 5. 구현 시 고려사항

### 5.1 프론트엔드
1. **동적 설문 폼 생성**
   - 관리자가 설정한 설문 항목을 동적으로 렌더링
   - 체크박스, 라디오 버튼, 텍스트 입력 등 다양한 입력 타입 지원

2. **설문 데이터 검증**
   - 필수 항목 입력 검증
   - 전화번호 형식 검증
   - 텍스트 길이 제한

### 5.2 백오피스
1. **약국 히스토리 팝업**
   - 모달 또는 사이드 패널 형태로 구현
   - 방문 이력을 시간순으로 정렬
   - 설문조사 내용 조회 기능

2. **관리자 비고 작성**
   - 인라인 편집 기능
   - 작성일시 및 작성자 기록

3. **데이터 관계 설계**
   - 약국 ↔ 방문기록 (1:N)
   - 방문기록 ↔ 설문응답 (1:1)
   - 방문기록 ↔ 피코프렌즈 (N:1)
