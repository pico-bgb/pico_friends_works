## 3. 주요 기능
1. **사용자 인증**
   - 로그인 (이메일 주소 / 비밀번호)
   - 회원가입 (이메일 주소, 비밀번호, 이름, 전화번호, 소속학교)

2. **업무 관리**
   - 배정된 업무 조회
   - 업무별 로그인 사용자 표시
   - 업무 상세 정보 (클릭 시 자세히 보기)
   - 미이미지 표시 기능

3. **약국 관리**
   - 약국 리스트 조회
   - 새 약국 추가
   - CSV 가져오기/내보내기
   - 약국 정보 수정 및 삭제

4. **업무 분배**
   - 업무 배정 및 분배
   - 약국 배정 및 선택
   - CSV 형태로 데이터 관리

5. **공지사항 관리**
   - 공지사항 리스트 조회 (최신 10개)
   - 공지사항 작성 및 상세보기
   - 리스트 수 제한: 10개
   - 노출항목: 번호, 제목, 작성일, 작성자

## 4. 기술 스택
- **Frontend**: Next.js
- **Backend**: Spring Boot
- **Database**: MySQL/PostgreSQL (권장)
- **인증**: JWT Token 기반 인증
