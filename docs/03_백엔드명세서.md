# PICOFriends Backend 명세서 (Spring Boot)

## 1. 기술 스택
- **Framework**: Spring Boot 3.x
- **Language**: Java 17+
- **Build Tool**: Gradle
- **ORM**: Spring Data JPA + Hibernate
- **Database**: MySQL 8.0+ / PostgreSQL 14+
- **Authentication**: Spring Security + JWT
- **Validation**: Spring Validation (Hibernate Validator)
- **API Documentation**: SpringDoc OpenAPI (Swagger)
- **File Processing**: Apache POI (CSV/Excel)
- **Logging**: SLF4J + Logback

## 3. Entity 설계

### 3.1 User (사용자)
```java
@Entity
@Table(name = "users")
public class User extends BaseEntity {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @Column(nullable = false, unique = true, length = 100)
    private String email;

    @Column(nullable = false)
    private String password;

    @Column(nullable = false, length = 50)
    private String name;

    @Column(nullable = false, length = 20)
    private String phoneNumber; // Format: 010-1234-5678

    @Column(nullable = false, length = 50)
    private String university; // 37개 대학 중 하나

    @Enumerated(EnumType.STRING)
    @Column(nullable = false)
    private UserRole role; // ADMIN, PICOFRIEND

    @Enumerated(EnumType.STRING)
    @Column(nullable = false)
    private UserStatus status; // PENDING, APPROVED, REJECTED

    @Column(columnDefinition = "TEXT")
    private String adminComment; // 관리자 코멘트

    @OneToMany(mappedBy = "user")
    private List<WorkAssignment> workAssignments;

    @OneToMany(mappedBy = "user")
    private List<VisitRecord> visitRecords;

    @OneToMany(mappedBy = "user")
    private List<RegistrationRecord> registrationRecords;
}

public enum UserRole {
    ADMIN, PICOFRIEND
}

public enum UserStatus {
    PENDING, APPROVED, REJECTED
}
```

### 3.2 Pharmacy (약국)
```java
@Entity
@Table(name = "pharmacies")

    @Id
```

### 3.4 WorkAssignment (업무 배정)
```java
@Entity
@Table(name = "work_assignments")
public class WorkAssignment extends BaseEntity {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @ManyToOne(fetch = FetchType.LAZY)
    @JoinColumn(name = "user_id", nullable = false)
    private User user;

    @ManyToOne(fetch = FetchType.LAZY)
    @JoinColumn(name = "work_id", nullable = false)
    private Work work;

    @ManyToOne(fetch = FetchType.LAZY)
    @JoinColumn(name = "pharmacy_id", nullable = false)
    private Pharmacy pharmacy;

    @Enumerated(EnumType.STRING)
    @Column(nullable = false)
    private AssignmentStatus status; // ASSIGNED, VISITED, REGISTERED, COMPLETED

    @Column
    private LocalDateTime visitedAt; // 방문 일시

    @Column
    private LocalDateTime registeredAt; // 회원가입 완료 일시

    @Column(length = 500)
    private String surveyUrl; // 구글 설문 URL

    @Column
    private Boolean surveyCompleted = false; // 설문 완료 여부
}

public enum AssignmentStatus {
    ASSIGNED,      // 배정됨
    VISITED,       // 방문 완료
    REGISTERED,    // 회원가입 완료
    COMPLETED      // 모든 작업 완료
}
```

## 6. Security 설정

### 6.1 SecurityConfig
```java
@Configuration
@EnableWebSecurity
public class SecurityConfig {

    @Bean
    public SecurityFilterChain securityFilterChain(HttpSecurity http) throws Exception {
        http
            .csrf().disable()
            .cors().and()
            .sessionManagement().sessionCreationPolicy(SessionCreationPolicy.STATELESS)
            .and()
            .authorizeHttpRequests()
                .requestMatchers("/api/auth/**").permitAll()
                .requestMatchers("/api/admin/**").hasRole("ADMIN")
                .anyRequest().authenticated()
            .and()
            .addFilterBefore(jwtAuthenticationFilter(), UsernamePasswordAuthenticationFilter.class);

        return http.build();
    }
}
```
