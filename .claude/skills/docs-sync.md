# 문서 동기화 스킬

문서 버전 간 차이점을 확인하고 docs_new/ 디렉토리를 업데이트합니다.

## 작업 수행

1. **현재 문서 버전 확인**
   - `docs_new/` 디렉토리의 모든 마크다운 파일 목록 조회
   - 각 문서의 마지막 수정 일시 확인
   - README.md의 문서 버전 확인

2. **화면설계서 PPT와 비교**
   - 최신 화면설계서 PPT 파일 식별 (파일명 날짜 기준)
   - 현재 프로젝트 루트에 있는 PPT 파일들:
     - `251030_피코프렌즈 화면설계서_v1.5(통합).pptx` (최신)
     - 하위 디렉토리: `피코프렌즈_v1.5/`의 슬라이드 이미지들
   - 슬라이드 이미지와 마크다운 문서의 화면 번호 매칭 확인

3. **일관성 검증**
   - 화면 번호 (PF-XXX, AD-XXX) 누락 확인
   - 공통 컴포넌트 (CC-XXX) 참조 일관성 확인
   - API 엔드포인트 명세와 실제 백엔드 코드 일치 여부 확인
   - 데이터베이스 스키마와 엔티티 클래스 일치 여부 확인

4. **동기화 리포트 생성**
   - 발견된 불일치 사항 목록화
   - 수정이 필요한 파일 및 위치 명시
   - 우선순위 부여 (Critical/High/Medium/Low)

5. **자동 수정 제안**
   - 단순 오타나 형식 문제는 자동 수정
   - 복잡한 내용 불일치는 사용자 확인 요청
   - 수정 사항을 Git diff 형식으로 표시

## 참조 파일

- 문서 디렉토리: `docs_new/`
- 화면설계서 이미지: `피코프렌즈_v1.5/`
- 백엔드 프로젝트: `/mnt/c/Dev/repo/pico_friends_be`
- 프론트엔드 프로젝트: `/mnt/c/Dev/REpo/pico_friends_fe`
