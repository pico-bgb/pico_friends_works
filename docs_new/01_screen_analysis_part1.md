### 2. PF-020: 로그인 후 화면 - 배정된 업무
**화면 구성:**
- 상단: 공지사항 검색바
- 로그인 후 화면
- 배정된 업무 섹션

**주요 기능:**
1. 공지사항
   - 관리자에서 작성한 공지사항이 롤링되어 보여짐
   - 클릭 시 단문 개시글이 10초의 시간 가중치 클릭되어 나타남
   - 공지사항 클릭 후 돌아가기 버튼(화면 업무로 이동)

2. 공지사항 팝업
   - 관리자가 작성한 공지사항이 모든 학생에게 노출되며, 이미지와 텍스트가 함께 노출됨
   - 버튼 클릭 시 화면 회피방식으로 팝업 기능으로 추가

3. 배정된 업무 리스트
   - 좌측 상단에 검색 필터
   - 날짜 선택 기능 (캘린더)
   - 학교별 카드 형식으로 표시
     - OOO 학교
     - 사업자번호: 000-0000-0000
     - 서울시 강남구 테헤란로 123
     - 000-0000-0000
     - "방문 인증" 버튼 (활성화)
     - "방문 완료" 버튼 (비활성화)

**디자인 노트:**
- 클릭된 항 을 표현하지 않고 추출된 표현하기로 결정

### 3. PF-030: 방문 인증 화면 (클릭 시)
**화면 플로우:**
1. 방문 인증 (클릭 시) → 모달 팝업
   - "사진 업로드 가능한 칸이 나타남"
   - "방문 사진 등록" 버튼

2. 동의 절차
   - "2025년 하반기 판매 동향 조사"
   - "대상 학교: {NAME}"
   - "학사님을 대상으로 현장 설문을 진행합니다."
   - "설문 조사" 버튼

3. 학교 정보 확인 화면
   - OOO 학교
   - 사업자번호: 000-0000-0000
   - 서울시 강남구 테헤란로 123
   - 000-0000-0000
   - "방문 완료" 버튼

**주요 노트:**
- 방문 인증이 완료되었다면 클릭이 비활성화 되고 자동으로 회색 업무 클릭이 하위로 가고기 원본으로 표기
- 업무 회원에 맞게 업무 클릭 표출이 보여지고, 하단으로 자동 클러어 기능으로 추가
- 드라마 회원에 맞게 업무 클릭 표출이 보여지면서, 하단으로 자동 클러어 기능도 추가로 지원
- 설문 조사 가능
  - 관리자에게 자동화 설문조사 기능
  - 관리자에게 자동화 설문조사 보호로 피코를 기업 시 방문 업무 가 기업 완료로 표기

### 4. 메모 (특이사항) 화면
**화면 구성:**
1. 리포트 기능
   - 구글 설문에 등록된 앱셀 파일의 학교 정보를 불러올 수 있는 것
   - 상기 각 학교 데이터를 불러온 것을 피코프렌즈가 기록하고 불러올 것

2. 각 문항을 피코프렌즈가 직접 작성
   - **설문 조사 양식(단락형, 선택형, 정량형)**
   - **설문 조사 데이터는 관리자에게 출제를 불 수 있어야 함**

**주요 설문 항목:**
1. 학사별 설명 (학교 텍스트 입력)
   - 학교 기본 정보 표시
   - 학사별 면담자 이름, 연락처 등 텍스트 직접 입력

2. 학교 운영 및 메모 사항
   - 온라인몰 이용 현황
   - HMP몰(온라인몰), 새로망, 단삼, 바로공
   - 공스트리트, JW삽, 기타
   - 주력 판매 상품군 (전문의약품, 일반의약품, 건강 기능 식품)
   - 주력 판매 제품 (텍스트 직접 입력)
   - 가장 시급하게 필요한 제품 (텍스트 입력)

3. 마케팅/홍보 관련 고민
   - 도/SNS 광고, 지역 홍보물, 구매링 기능 상품 등
   - 기타 (텍스트 직접 입력)

4. 피코프렌즈 방응
   - 피코를 해봄 언냐 후 방용 (매우 좋음 ~ 매우 나쁨)
   - 주력 판매 상품군

5. 현장활동 요약 및 다음 단계 계획
   - 전체 현장활동 흐름에서 택스트 인력

**다음페이지 계속:**
6. 마케팅/홍보 관련 고민(래크박스, 중복 가능, 기타(택스트 입력))
7. 피코프렌즈 방응(라디오박스, 기타 택스트 입력)
8. 현장활동 요약 및 다음 단계 계획

### 5. PF-040: 마이페이지 클릭 후 화면
**화면 구성:**
1. 대시보드 기능
   - 대시보드의 숫자들은 추후 가입 학교 자동 코칭
   - 관더어트는 매 5초 이상 시 자동 직접
   - 이미지 기능
   - 편리가가 자장한 이미지를 노출 한다

2. 행잠브드(Leaderboard)
   - 피코프렌즈 오늘의 방문 1위 ~ 10위 화면
   - 랭킹 순위가 정해지며 등급자가 있을 경우 누락임
   - (만약 순위가 3명이라면 3위까지 ** 표기)에 컵게 표기하면 된
   - 하단 데이터 취합 기능을 텍스트로 고차한다

**피코프렌즈 행잠브드 운영 규칙:**
1. 행잠브드 전체 기준
   - 랭킹 기준: 방문 횟수, 10초+ (1등자의 증감도 반영)
   - 분류: 관리 네임(Sort): 피코프렌즈 닉네임으로 코기 하며 클릭하면 프로필 순위
   - 이름(Name): 피코프렌즈 닉네임(로그인 아이디 클릭자 화자명 *** 표기)
   - 순위 범 등(Rate of Change): 지난주 대비 순위 등장 (확살)
   - 회원가입 학교수(Pharm): 해당 주차 피코를 기능 산출한 학교

2. 종합 산정 기준 (KPI)
   - 기본 방문 (Visit): CRM 기준학 방운 로그,가적당 10점
   - 학점 앱문가점 선 정(Sign): 피코를 신규 가입,가적당 100점

3. 점수 샐정 기준 (KPI)
   - 4. 순위 변동 (증감) 표기 방식
   - 기출은 방문(OOO기) 표기, 1주일 전 1위일 시, 순위 ▼
   - 증전 합계 증감
   - 지난주 대비 순위 증감 (같은 순위 유지 시 순위 변동 NEW!)
   - 순위 변동 증감 - NEW: 행잠브드 신규 전입 변동 (1주차 활동 사작 시 전체 NEW!)

4. 순위 변동 (증감) 표기 기능
5. 등장가 취급 기능
   - 순위가 동점일 경우, 순풍 신규 회원가입 점수 > 순풍 방문 점수 순으로 달러
