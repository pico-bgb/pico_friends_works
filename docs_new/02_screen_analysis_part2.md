# 피코프렌즈 화면설계서 분석 (Part 2 - 백오피스)

## 문서 정보
- 버전: Ver 1.4
- 범위: 슬라이드 11-26 분석 결과 (백오피스 Web 화면)
- 분석일: 2025-10-27

## 백오피스 화면 ID 범위
- Web Admin: AD-010 ~ AD-074

## 백오피스 메뉴 구조

```
홈
├── 대시보드
├── 사용자관리
├── 약국관리
├── 업무분배
├── 계시판관리
└── 통계
```

## 분석된 화면 목록

### 1. AD-020: 대시보드 (홈)

**화면 구성:**
- 상단 네비게이션: 대시보드, 사용자관리, 약국관리, 업무분배, 계시판 관리, 통계
- 로그아웃 버튼

**주요 위젯:**
1. 학교 방문 통계 (전체)
   - 학교 방문(전체): OO
   - 회원 가입(전체): OO
   - 전환율(전체): OO%

2. 학교 방문 통계 (오늘)
   - 학교 방문(오늘): OO
   - 회원 가입(오늘): OO
   - 전환율(오늘): OO%

3. 전체 전적률
   - 프렌즈별 활동 순위 표시

4. 학교별 성과
   - 그래프 및 차트로 시각화

**Description:**
1. 관리자 로그인 후 홈 화면 (관리자 AD-010 로그인 화면은 PF-010과 유사함)
2. 피코프렌즈 전체, 각 과목 수 - 전체 학교 가입 수, 방문횟수, 신규 활동(%) (학교 가입 수 = 전체 학교 중 수) x 100
3. 오늘 학교 방문, 오늘 회원 가입, 오늘 전환율(%)
4. 피코프렌즈 전체 활동 등록률 그래프
   - 데이터는 학교 방문 회원가입 기준으로 되어야 따라서 그래프로 범위로로 표기하고, 월차, 주차, 출간 조회로 등기
5. 전체 전적률 - 오늘 전적 통계 그래프
   - 리스트 노출 수: 10개
   - 리스트 노출 순서: 직접 등록으로 노출
6. 일부 데이터 빠려주고, 상세의 강우 Redash 활용

### 2. AD-030: 사용자 관리

**화면 구성:**
1. 조건부 검색 필터
   - 역할: 전체직원, 관리자, 관리자, 피코프렌즈
   - 상태: 전체보기, 승인대기, 승인, 보류, 승인취소, 승인 거부
   - 이름으로 검색

2. 사용자 등록
   - CSV 다운로드
   - CSV 내보내기

3. 사용자 리스트 테이블
   - 컬럼: 이름, 이메일(ID), 전화번호, 소속, 상태, 피코프렌즈, 히스토리, 관리
   - 관리 버튼: 승인(녹색), 거부(빨간색), 삭제(휴지통 아이콘)
   - 페이지네이션

4. 사용자 등록 가입 승인/거부
   - 버튼 형태의 승인/거부 절차 불수반되어 승인되어 가능함

5. 수정(관리자가 직접 승인/거부)
   - 삭제(사용자 정보 삭제 가능)

**사용자 등록/수정 모달:**
- 사용자 등록 모달 (오른쪽)
  - 이메일
  - 비밀
  - 이름
  - 전화
  - 소속
  - "사용자 등록" 버튼

- 사용자 수정 모달
  - 동일한 필드
  - "사용자 수정" 버튼

**Description:**
1. 조건부 검색 필터
   - 역할: 관리자, 관리자자, 피코프렌즈, 승인대기, 승인대기부
2. 사용자 등록
   - 사용자의 경우 관리자, 부여, 피코프렌즈 구분
3. CSV 다운로드
   - 검색 된 내역는 사용자 정보 다운로드
4. 사용자 회원 가입 승인/거부
   - 버튼 형태의 승인/거부 불수반되어 등록되어 가능함
5. 수정(관리자가 직접 승인/거부)
   - 삭제(사용자 정보 삭제 가능)

### 3. AD-040: 약국 관리

**화면 구성:**
1. 새 학교 추가 시 데이터를 입력하여 등록
   - CSV 가적으로 피일을 시스템에 등록 가능
   - CSV 내보내기로 등록된 학교 리스트를 내보냄

2. 약국 리스트 테이블
   - 컬럼: 이름, 주소, 전화번호, 히스토리, 관리
   - 히스토리 버튼으로 방문 기록 확인
   - 관리 컬럼: 삭제 가능

3. 히스토리를 확인 할 수 있음
   - 비고는 현 업무에게 클릭 시 이야기가 할 수 없음

4. 비고는 관리자가 코멘트를 직접 작성

5. 약국 정보 수정 및 삭제
   - 데이터를 정보를 수정 및 삭제 가능

**약국 등록/수정 모달:**
- 약국 등록 (오른쪽)
  - 이름
  - 주소
  - 전화번호
  - "약국 등록" 버튼

- 약국 정보 수정
  - 동일 필드
  - "약국 정보 수정" 버튼

**Description:**
1. 새 학교 추가 시 데이터를 입력하여 등록
2. CSV 버전
   - CSV 작성의 파일로 시스템에 등록
   - CSV 내보내기
   - 등록된 학교 리스트를 내보냄
3. 히스토리를 확인 할 수 있음
   - 비고는 현 업무에게 클릭 시 이야기를 확인 할 수 있음
4. 비고는 관리자가 코멘트를 직접 작성

### 4. AD-044: 약국 방문 히스토리

**화면 구성:**
- 모달 형태로 표시
- 제목: "약국 방문 히스토리"
- 약국 정보:
  - 약국 이름: OOOO약국
  - 약국 사업자번호: 000-00-00000
  - 약사 이름: OOO
  - 약국 연락처: 00-000-0000

**방문 히스토리 테이블:**
- 컬럼:
  - 방문 날짜
  - 회원 가입 날짜
  - 피코프렌즈
  - 히스토리
  - 비고
  - 관리
- 히스토리 아이콘: 각 방문 기록 상세 확인
- 비고: 관리자가 직접 입력 (클릭 시 입력)
- 관리 컬럼: 수정(연필 아이콘), 삭제(휴지통 아이콘)

**Description:**
1. 방문의 아이콘 클릭 시 코드프렌즈 방운에 대한 가장 최근 히스토리가 표시 외 시하는 아이콘 관리
2. 피코프렌즈가 가학한 날자 니모니, 회원가입을 하지 않았을 영 울려도 가능하게 기록
3. 피코프렌즈가 작성한 설문조사는 클릭 시, 설문조사 작성 내용을 확인 수 있어야 함 (1~1)
4. 비고의 승랙 관리자가 직접 텍스트로 입력 할 수 있음
   - 텍스트는 한줄 내 작업 가능

### 5. AD-050: 업무 분배

**화면 구성:**
1. 조건부 검색 필터
   - 약국: 이름으로 검색
   - 상태: 전체보기, 배정, 미배정
   - 사용자 이름으로 검색

2. [NUM] 업무 배분 - 선택한 약국 개수 표시

3. 업무 리스트 테이블
   - 체크박스로 다중 선택 가능
   - 컬럼: 약국이름, 주소, 전화번호, 상태, 피코프렌즈, 히스토리, 비고
   - 상태: "배정됨" (파란색 배지)
   - 페이지네이션

4. "피코프렌즈 업무 배분" 모달 (오른쪽)
   - 피코프렌즈 이름 검색 필드
   - 체크박스 리스트로 선택
     - 이름, 학교
   - 취소/확인 버튼

**Description:**
1. 조건부 검색 필터
   - 약국, 전체/배정, 상태, 전체보기, 미배정
   - 사용자 이름으로 검색
2. CSV 가적의 자적된 클릭로 업무 분배 올 CSV 활전로 다운로드
3. 제크박스에 체코를 하면 체코 된 결과를 순차가 나타나고 클릭 시 3-1 팝업이 노출되며, 업무를 활정할 수 있음
4. 동록된 피코프렌즈 검색 및 업무 분배
5. 히스토리의 업무 10명이야, 히스토리가 나타남
6. 비고는 관리자가 작업 코멘트를 입력하고 수정 할 수 있음

### 6. AD-060: 계시판 관리

**화면 구성:**
1. 공지 관리 글 목록 서 게정 내용 상세보기 화면으로 이동
   - 리스트 수: 10개
   - 노물결과: 번호/제목/작성일/상태/작성자
   - 리스트 노출 순서: 최신 순으로 노출

2. 분류 내 이미지 준비 시 카메라 아이콘 노출

3. 내용 클릭 시 공지 작성 3-1로 이동
   - 기능 자시어 공지 공지로포 사항

4. "공개사항 작성하기" 버튼

**계시판 리스트 테이블:**
- 컬럼: 번호, 제목, 작성일, 작성자
- 페이지네이션 (1-10 페이지)
- 제목 수+나별 검색 기능

**공지사항 작성/수정 화면 (오른쪽):**
- 필드:
  - 카테고리 (드롭다운)
  - 작성자
  - 제목
  - 내용 (리치 텍스트 에디터 - 텍스트 포맷팅 도구 포함)
- 목록보기 / 등록하기 버튼

**Description:**
1. 공지 관리 글 목록 서 게정 내용 상세보기로 이동
   - 리스트 수: 10개
   - 리스트 노출 순서: 최신 순으로 작성 10개를 기록하고 노출
2. 분류 내 이미지 준비 시 카메라 아이콘 노출
3. 내용 클릭 시 공지 작성 3-1로 이동(기능: 피코프렌즈 모든에 대한 기능이 공지가이아니라 관리 취할하며 이미지 관리) 공지사항 클릭 박 구 엄시 출제별 불수 있어야 함 (1~1)
4. 댓글 가능을 통해 관리자 업무 분배 여구 확인
5. 설문조사 관리로 이동

### 7. AD-070: 통계

**화면 구성:**
1. 조건부 검색 필터
   - 기간: 지역/서/공/등 표기), 약국명, 피코프렌즈, 방문, 회원가입 날짜
   - 다운로드: 간은 전체

2. 기간 선택 (캘린더)
   - 시작일~종료일 선택
   - 전체 / 방문안료 / 회원가입 / 내용 필터

3. CSV 내보내기 버튼

4. 통계 테이블
   - 컬럼: 이름, 지역, 약국, 방문날짜, 설문 완료 시간, 상태, 월급, 히스토리, 비고
   - 상태: "방문안료", "회원가입"
   - 월급: "오늘", "추간", "주간"
   - 달성률: 00%
   - 히스토리 아이콘
   - 페이지네이션

5. 하단 통계 바
   - 각 사용자별 통계
   - 총 배정 수, 방문안료 수, 회원가입 수, 달성률

**Description:**
1. 조건부 검색 필터
   - 기준, 지역(시/군/등 표기), 약국명, 피코프렌즈, 방문, 회원가입 날짜
   - 다운로드 간은 전체
2. 히스토리의 업무 10명이야지 텍스트를 입력 민 수 있음
   - 비고의 승랙 관리자가 텍스트를 입력 민 수 있음 (1-1)
3. 담성률 = 회원가입 수 / 총 배정 수
4. 검색 결과를 CSV 형태로 다운로드
5. 내역이 많을 경우 다음 페이지로 이동

## 종합 정리

### 백오피스 시스템 주요 기능

1. **대시보드**: 전체 통계 및 KPI 모니터링
2. **사용자 관리**: 피코프렌즈 및 관리자 계정 관리, 승인/거부 프로세스
3. **약국 관리**: 약국 정보 CRUD, 방문 히스토리 추적
4. **업무 분배**: 피코프렌즈에게 약국 방문 업무 할당
5. **계시판 관리**: 공지사항 작성 및 관리
6. **통계**: 상세 필터링 및 데이터 추출 기능

### 공통 기능
- CSV Import/Export
- 페이지네이션
- 검색 및 필터링
- 모달 기반 CRUD 작업
